# ECE-309-Final-Project
### ECE 309 Final Project - Fall 2022
Created by Joseph Homenick, Trey Parker, Jack Maxwell, Nick Urch
<br>
<br>
This is a program that implements the board game *Sorry!* in C++. The program implements the standard *Sorry!* rules.
<br>
<br>
To compile and run the program, download all files in the repository. Compile the files and run the main.cpp file
<br>
to start playing!
<br>
<br>
The game can be played with both human and nonhuman players. The program also displays the 'physical' board in the terminal window, allowing players to see the real-time status of the board and their pawns.
<br>
<br>
Included in this repository is the written report.
